1. What is Semantic HTML? 
 - Elements used to describe the meaning to the browser and developer.
2. What is HTML used for?
 - HTML is the code used to structure a web page and its content.
3. What is an attribute and where do we put it? 
 - Special words used inside of the opening tag to control the element's behaviour.
4. What is the h1 tag used for? How many times should I use it on a page?
 - Indicates the heading on a website. The most important header. Should only be used once. 
5. Name two tags that have required attributes
 - The <img /> and <a> tags.
6. What do we put in the head of our HTML document? 
 - Metadata which is not displayed. Metadata defines the document title, character set, styles, scripts, and other meta info.
7. What is an id? 
 - HTML attribute used to specify a unique id for an HTML element. The value of an ID msut be unique within the HTML document.

 - Any element when needed.

10. What is a class? 
 - Attribute in HTML used to specify a class for an HTML element. 
11. What elements can I add a class to?

12. How many times can I use the same class on a page? 
 - As many times as needed.

 - adding the "target" attribute and setting it to "blank"
14. What is the alt attribute in the image tag used for? 
 - Adds a text description to an image. ALT text is accessed by screen reader users to provide them with a text equivalent of images.

22. What is a parent child selector? When would this be useful? 
 - A parent child selector selects all elements that are a direct child of the specified element. Useful for CSS specificity

24. What is a pseudo selector?
 - A pseudo-class is a selector that selects elements that are in a specific state, e.g. they are the first element of their type, or they are being hovered over by the mouse pointer.
